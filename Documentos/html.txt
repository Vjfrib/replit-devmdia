Para substituir todas as variáveis com o mesmo nome, você pode usar a combinação de teclas CTRL + E no Replit. Isso irá selecionar todas as instâncias de uma variável ou palavra que você queira substituir.

Para substituir todas as instâncias de um nome por outro, você pode usar a combinação de teclas CTRL + H, que irá abrir uma janela de pesquisa e substituição de texto no editor.

Para organizar o código, você pode usar a funcionalidade "formatar código automaticamente" no Replit IDE. Para fazê-lo, basta pressionar as teclas "Ctrl + Shift + F" no Windows ou "Cmd + Shift + F" no MacOS.

Para ver a lista completa de atalhos do Replit IDE, você pode ir para o menu "Ajuda" na barra superior e clicar em "Atalhos do teclado" ou pressionar "Ctrl + /" no Windows ou "Cmd + /" no MacOS para obter a lista de atalhos rapidamente.
